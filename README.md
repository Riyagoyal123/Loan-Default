💳 Loan Default Analysis Dashboard | Power BI
Project Overview
This project presents an interactive Loan Default Analysis Dashboard developed in Power BI using a loan dataset stored in Microsoft Excel. The project focuses on analyzing borrower demographics, financial profiles, loan distribution, and default patterns to uncover insights that can support better lending decisions and risk assessment.
The dataset was imported into Power BI, where extensive Power Query transformations were performed to clean and prepare the data. Custom DAX measures were created (organized in Measure Tables 1, 2, and 3) to calculate KPIs, financial metrics, time intelligence measures, and other analytical insights. The final dashboard provides an intuitive and interactive way to explore loan performance and default trends.
Project Workflow
1. Data Import
Imported the loan dataset from Microsoft Excel into Power BI.
2. Data Preparation (Power Query)
Cleaned and transformed the dataset using Power Query.
Managed missing values and corrected data inconsistencies.
Created calculated columns where required.
Optimized the data model for reporting.
3. Data Modeling & DAX
Built relationships between tables.
Developed DAX Measures in Measure Tables 1, 2, and 3.
Created KPIs, calculated metrics, and time intelligence measures for detailed business analysis.
4. Dashboard Development
Designed interactive dashboards using Power BI.
Added slicers, filters, KPI cards, charts, and drill-down capabilities for enhanced user interaction.
